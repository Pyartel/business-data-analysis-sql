# Cohort Retention Analysis

## Project Overview

This project analyzes user retention across organic and promotional acquisition channels using SQL and cohort analysis techniques.

## Business Goal

Evaluate long-term retention performance of different acquisition channels and identify opportunities to improve customer lifetime value.

## Technologies

- SQL
- PostgreSQL
- Cohort Analysis
- Retention Analysis
- Excel
- Data Visualization

## Key Findings

- Organic users demonstrated stronger long-term retention.
- Promotional cohorts showed faster churn rates.
- Retention differences increased over time.
- Several recommendations were developed to improve acquisition efficiency.
